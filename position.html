Using position:Static:

Static

Static is the default position in CSS. The element stays in its normal place in the document flow and does not move even if we try to change its position.

Using position:Absolute:

Absolute

Absolute position removes the element from normal flow. It is placed relative to its nearest parent and can be moved anywhere.

Using position:Sticky:

Sticky

Sticky position is a mix of relative and fixed. It stays normal until you scroll, then it sticks to the top of the screen.

Using position:Relative:

Relative

Relative position moves the element from its original place. Other elements do not shift, and the original space is still reserved.

Using position:Fixed:

Fixed

Fixed position keeps the element in the same place on the screen even when scrolling. It is always visible and does not move with the page.
